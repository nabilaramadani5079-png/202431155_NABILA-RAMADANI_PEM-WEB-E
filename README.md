Link Hasil Deploy (GitHub Pages):

Tugas ini bertujuan untuk mengimplementasikan konsep Responsive Web Design (RWD) melalui pembuatan website portofolio pribadi multi-halaman.
Website ini diharapkan dapat menampilkan identitas, keterampilan, serta proyek-proyek mahasiswa secara menarik, interaktif, dan dapat diakses dengan baik di berbagai ukuran layar (desktop, tablet, maupun ponsel).

Struktur Halaman Website
1. Halaman Home / Beranda (index.html)
   Halaman utama yang berfungsi sebagai pengenalan diri mahasiswa.
   Bagian-bagian yang harus ada:
   - Header & Navigasi;Navigasi utama yang konsisten di seluruh halaman (Home, Portofolio, dan Kontak).
   - Hero Section: Menampilkan foto diri dan deskripsi singkat sebagai pengantar profil pribadi.
   - About Me: Berisi informasi mengenai latar belakang, pendidikan, minat, dan kepribadian.
   - Keterampilan/Keahlian: Menampilkan kemampuan utama
